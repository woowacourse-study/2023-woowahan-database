# 33강 - 관계형 모델

## 1) 관계형 모델

관계형 모델의 기본적인 요소는 **릴레이션(Relation)**

> 릴레이션은 테이블을 말한다!

**릴레이션의 특징**

- 하나 이상의 관계를 바탕으로 연산한다.
- 연산한 결과, 반환되는 것 또한 관계이다.
- 연산을 중첩 구조로 실행해도 상관없다.

## 2) 관계형 모델과 SQL

### 합집합

- UNION
- 릴레이션끼리의 덧셈

### 차집합

- EXCEPT
- 릴레이션끼리의 뺄셈

### 교집합

- INTERSECT
- 릴레이션끼리의 공통부분

### 곱집합

- CROSS JOIN
- 릴레이션끼리의 대전표 연산

### 선택

- WHERE
- 튜플의 추출, 데이터를 검색하는 것

### 투영

- SELECT
- 반환할 열을 지정하는 것

### 결합

- 릴레이션끼리 교차결합해 계산된 곱집합에 결합조건을 만족하는 튜플 추출
